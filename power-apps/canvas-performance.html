Canvas App Performance Optimization

This guide focuses on improving performance in Power Apps canvas apps, especially for enterprise-scale applications.

Common Performance Issues

  • Delegation warnings
  • Excessive API calls
  • Heavy formulas on screen load

Best Practices

Tip: Avoid using non-delegable functions on large datasets.
  • Use delegation-friendly filters
  • Limit OnStart logic
  • Cache data using collections

When to Re-architect

If performance issues persist, consider splitting apps or redesigning data access patterns.
